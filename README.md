# FuckBuild
a minimal build system for `gcc`/`g++`, written in bash. \
(technically *not* a build system at all, just a wr\*\*\*\*r, we don't say the **w-word**..)

## usage
syntax: `fbd [language] [input] [output] [compiler_parameters]` \
`language` = `C` or `C++` \
`input` = input filename **without file extension** \
`output` = output filename \
`compiler_parameters` = any extra compiler parameters (i.e. `-lncurses`, `-Og`)

#### ..that or just run `fbd` and it'll guide you through the compilation
